# Floyd Desktop

## Overview
Floyd Desktop is an AI Assistant designed for desktop use through a web interface. This project enables users to interact with various AI features in a seamless manner.

## Features
- AI Integration: Leverages various AI services for an enhanced user experience.
- Responsive Design: Mobile-friendly design compatible with all major devices.
- Modular Architecture: Easily extendable and maintainable code structure.

## Getting Started
To get started with the Floyd Desktop project, follow the instructions below:

### Prerequisites
- Node.js (version 16.x or later)
- npm (Node package manager)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/floyd-desktop.git
   ```
2. Navigate to the project directory:
   ```bash
   cd floyd-desktop
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project
To start the development server,
```bash
npm run dev
```
This command will run both the server and client.

### Building for Production
To build the project for deployment,
```bash
npm run build
```

## Technologies Used
- React for front-end UI
- Express.js for backend services
- TypeScript for static typing
- Tailwind CSS for styling
